# MindBlog - Frontend

Interface do usuário da plataforma MindBlog, desenvolvida para o case da Mind Consulting.

## Tecnologias
- React 19
- TypeScript
- Vite
- React Router DOM v7
- Lucide React (Ícones)
- Vanilla CSS (Design System escuro moderno)

## Funcionalidades
- Feed de artigos com filtros por categoria e busca por palavras-chave
- Leitura individual de artigos com barra de progresso de leitura no topo
- Sistema de comentários com edição e remoção restritas ao próprio autor
- Curtidas e salvamento (bookmarks) por usuário
- Painel Dashboard com métricas de engajamento do usuário logado
- Tela de configurações para atualização de perfil e envio de foto de avatar

## Instalação e Execução

```bash
# Instalar dependências
npm install

# Rodar em modo de desenvolvimento
npm run dev
```

Acesse `http://localhost:5173` no seu navegador.
